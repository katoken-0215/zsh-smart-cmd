== Repository launch commands

ghq + skim (sk) でリポジトリを選び、ターミナルまたは Claude Code で開く。

new-term::
    ghq リポジトリを skim (sk) で選び、そのディレクトリで Ghostty を開く。

new-cc::
    ghq リポジトリを skim (sk) で選び、cmux の新規ワークスペースで Claude Code を起動する。

new::
    skim (sk) で `terminal (Ghostty)` / `claude-code (cmux)` を選び、new-term / new-cc に委譲する。
